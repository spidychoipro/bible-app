# 성경 읽기 (개역한글)

개역한글 성경 PWA 앱. 구약 39권, 신약 27권, 오프라인 지원.

## 기능

- 구약/신약별 책 목록 → 장 선택 → 절 읽기
- 특정 구절 딥링크: `#book=창세기&ch=1&v=1`
- 본문 검색
- 다크모드
- PWA 설치 가능 (오프라인 지원)

## 사용하기

**GitHub Pages (추천):**
